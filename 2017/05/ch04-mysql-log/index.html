дәҢиҝӣеҲ¶ж—Ҙеҝ— binary logВ¶

дәҢиҝӣеҲ¶ж—Ҙеҝ— (binary log)пјҡз”ЁжқҘи®°еҪ•mysqlеҶ…йғЁеўһеҲ ж”№зӯүеҜ№mysqlж•°жҚ®еә“жңүжӣҙж–°зҡ„еҶ…е®№зҡ„и®°еҪ•пјҲеҜ№ж•°жҚ®еә“зҡ„ж”№еҠЁпјүпјҢеҜ№ж•°жҚ®еә“жҹҘиҜўзҡ„иҜӯеҸҘеҰӮпјҡshowгҖҒselectејҖеӨҙзҡ„иҜӯеҸҘпјҢдёҚдјҡиў«binlogж—Ҙеҝ—и®°еҪ•пјҢз”ЁдәҺж•°жҚ®еә“зҡ„еўһйҮҸжҒўеӨҚпјҢд»ҘеҸҠдё»д»ҺеӨҚеҲ¶

binlogдёүз§ҚжЁЎејҸВ¶

Row Level

дёҚи®°еҪ•sqlиҜӯеҸҘдёҠдёӢж–Үзӣёе…ідҝЎжҒҜпјҢд»…дҝқеӯҳе“ӘжқЎи®°еҪ•иў«дҝ®ж”№гҖӮ

  • дјҳзӮ№пјҡbinlogдёӯеҸҜд»ҘдёҚи®°еҪ•жү§иЎҢзҡ„sqlиҜӯеҸҘзҡ„дёҠдёӢж–Үзӣёе…ізҡ„дҝЎжҒҜпјҢд»…йңҖиҰҒи®°еҪ•йӮЈдёҖжқЎи®°еҪ•иў«дҝ®ж”№жҲҗд»Җд№ҲдәҶгҖӮжүҖд»Ҙrowlevelзҡ„ж—Ҙеҝ—еҶ…е®№дјҡйқһеёёжё…жҘҡзҡ„и®°еҪ•дёӢжҜҸдёҖиЎҢж•°жҚ®дҝ®ж”№зҡ„з»ҶиҠӮгҖӮиҖҢдё”дёҚдјҡеҮәзҺ°жҹҗдәӣзү№е®ҡжғ…еҶөдёӢзҡ„еӯҳеӮЁиҝҮзЁӢпјҢжҲ–functionпјҢд»ҘеҸҠtriggerзҡ„и°ғз”Ёе’Ңи§ҰеҸ‘ж— жі•иў«жӯЈзЎ®еӨҚеҲ¶зҡ„й—®йўҳ

  • зјәзӮ№пјҡжүҖжңүзҡ„жү§иЎҢзҡ„иҜӯеҸҘеҪ“и®°еҪ•еҲ°ж—Ҙеҝ—дёӯзҡ„ж—¶еҖҷпјҢйғҪе°Ҷд»ҘжҜҸиЎҢи®°еҪ•зҡ„дҝ®ж”№жқҘи®°еҪ•пјҢиҝҷж ·еҸҜиғҪдјҡдә§з”ҹеӨ§йҮҸзҡ„ж—Ҙеҝ—еҶ…е®№,жҜ”еҰӮдёҖжқЎupdateиҜӯеҸҘпјҢдҝ®ж”№еӨҡжқЎи®°еҪ•пјҢеҲҷbinlogдёӯжҜҸдёҖжқЎдҝ®ж”№йғҪдјҡжңүи®°еҪ•пјҢиҝҷж ·йҖ жҲҗbinlogж—Ҙеҝ—йҮҸдјҡеҫҲеӨ§пјҢзү№еҲ«жҳҜеҪ“жү§иЎҢalter tableд№Ӣзұ»зҡ„иҜӯеҸҘзҡ„ж—¶еҖҷпјҢз”ұдәҺиЎЁз»“жһ„дҝ®ж”№пјҢжҜҸжқЎи®°еҪ•йғҪеҸ‘з”ҹж”№еҸҳпјҢйӮЈд№ҲиҜҘиЎЁжҜҸдёҖжқЎи®°еҪ•йғҪдјҡи®°еҪ•еҲ°ж—Ҙеҝ—дёӯгҖӮ

Statement Level

жҜҸдёҖжқЎдјҡдҝ®ж”№ж•°жҚ®зҡ„sqlйғҪдјҡи®°еҪ•еңЁbinlogдёӯгҖӮ

  • дјҳзӮ№пјҡдёҚйңҖиҰҒи®°еҪ•жҜҸдёҖиЎҢзҡ„еҸҳеҢ–пјҢеҮҸе°‘дәҶbinlogж—Ҙеҝ—йҮҸпјҢиҠӮзәҰдәҶIOпјҢжҸҗй«ҳжҖ§иғҪгҖӮ(зӣёжҜ”rowиғҪиҠӮзәҰеӨҡе°‘жҖ§иғҪдёҺж—Ҙеҝ—йҮҸпјҢиҝҷдёӘеҸ–еҶідәҺеә”з”Ёзҡ„SQLжғ…еҶөпјҢжӯЈеёёеҗҢдёҖжқЎи®°еҪ•дҝ®ж”№жҲ–иҖ…жҸ’е…Ҙrowж јејҸжүҖдә§з”ҹзҡ„ж—Ҙеҝ—йҮҸиҝҳе°ҸдәҺStatementдә§з”ҹзҡ„ж—Ҙеҝ—йҮҸпјҢдҪҶжҳҜиҖғиҷ‘еҲ°еҰӮжһңеёҰжқЎд»¶зҡ„updateж“ҚдҪңпјҢд»ҘеҸҠж•ҙиЎЁеҲ йҷӨпјҢalterиЎЁзӯүж“ҚдҪңпјҢROWж јејҸдјҡдә§з”ҹеӨ§йҮҸж—Ҙеҝ—пјҢеӣ жӯӨеңЁиҖғиҷ‘жҳҜеҗҰдҪҝз”ЁROWж јејҸж—Ҙеҝ—ж—¶еә”иҜҘи·ҹжҚ®еә”з”Ёзҡ„е®һйҷ…жғ…еҶөпјҢе…¶жүҖдә§з”ҹзҡ„ж—Ҙеҝ—йҮҸдјҡеўһеҠ еӨҡе°‘пјҢд»ҘеҸҠеёҰжқҘзҡ„IOжҖ§иғҪй—®йўҳгҖӮ)

  • зјәзӮ№пјҡз”ұдәҺи®°еҪ•зҡ„еҸӘжҳҜжү§иЎҢиҜӯеҸҘпјҢдёәдәҶиҝҷдәӣиҜӯеҸҘиғҪеңЁslaveдёҠжӯЈзЎ®иҝҗиЎҢпјҢеӣ жӯӨиҝҳеҝ…йЎ»и®°еҪ•жҜҸжқЎиҜӯеҸҘеңЁжү§иЎҢзҡ„ж—¶еҖҷзҡ„дёҖдәӣзӣёе…ідҝЎжҒҜпјҢд»ҘдҝқиҜҒжүҖжңүиҜӯеҸҘиғҪеңЁslaveеҫ—еҲ°е’ҢеңЁmasterз«Ҝжү§иЎҢж—¶еҖҷзӣёеҗҢ зҡ„з»“жһңгҖӮеҸҰеӨ–mysql зҡ„еӨҚеҲ¶,еғҸдёҖдәӣзү№е®ҡеҮҪж•°еҠҹиғҪпјҢslaveеҸҜдёҺmasterдёҠиҰҒдҝқжҢҒдёҖиҮҙдјҡжңүеҫҲеӨҡзӣёе…ій—®йўҳ(еҰӮsleep()еҮҪж•°пјҢ last_insert_id()пјҢд»ҘеҸҠuser-defined functions(udf)дјҡеҮәзҺ°й—®йўҳ).

Mixed Level

жҳҜд»ҘдёҠдёӨз§Қlevelзҡ„ж··еҗҲдҪҝз”ЁпјҢдёҖиҲ¬зҡ„иҜӯеҸҘдҝ®ж”№дҪҝз”Ёstatmentж јејҸдҝқеӯҳbinlogпјҢеҰӮдёҖдәӣеҮҪж•°пјҢstatementж— жі•е®ҢжҲҗдё»д»ҺеӨҚеҲ¶зҡ„ж“ҚдҪңпјҢеҲҷйҮҮз”Ё row ж јејҸдҝқеӯҳbinlogпјҢMySQLдјҡж №жҚ®жү§иЎҢзҡ„жҜҸдёҖжқЎе…·дҪ“зҡ„sqlиҜӯеҸҘжқҘеҢәеҲҶеҜ№еҫ…и®°еҪ•зҡ„ж—Ҙеҝ—еҪўејҸпјҢд№ҹе°ұжҳҜеңЁStatementе’ҢRowд№Ӣй—ҙйҖүжӢ©дёҖз§ҚгҖӮж–°зүҲжң¬зҡ„MySQLдёӯйҳҹrow levelжЁЎејҸд№ҹиў«еҒҡдәҶдјҳеҢ–пјҢе№¶дёҚжҳҜжүҖжңүзҡ„дҝ®ж”№йғҪдјҡд»Ҙrow levelжқҘи®°еҪ•пјҢеғҸйҒҮеҲ°иЎЁз»“жһ„еҸҳжӣҙзҡ„ж—¶еҖҷе°ұдјҡд»ҘstatementжЁЎејҸжқҘи®°еҪ•гҖӮиҮідәҺupdateжҲ–иҖ…deleteзӯүдҝ®ж”№ж•°жҚ®зҡ„иҜӯеҸҘпјҢиҝҳжҳҜдјҡи®°еҪ•жүҖжңүиЎҢзҡ„еҸҳжӣҙгҖӮ

  2. е…¬еҸёеҰӮжһңз”ЁеҲ°дҪҝз”ЁMySQLзҡ„зү№ж®ҠеҠҹиғҪпјҲеӯҳеӮЁиҝҮзЁӢгҖҒи§ҰеҸ‘еҷЁгҖҒеҮҪж•°пјүгҖӮеҲҷйҖүжӢ©MixedгҖӮ
  3. е…¬еҸёеҰӮжһңе…¬йҒ“дҪҝз”ЁMySQLзҡ„зү№ж®ҠеҠҹиғҪпјҲеӯҳеӮЁиҝҮзЁӢгҖҒи§ҰеҸ‘еҷЁгҖҒеҮҪж•°пјүпјҢеҸҲеёҢжңӣж•°жҚ®жңҖеӨ§еҢ–дёҖиҮҙпјҢжӯӨж—¶жңҖеҘҪз”ЁrowжЁЎејҸ
