ELK 收集syslog

cylon 收录于 ELK
 2018-11-02 00:00:00  约 192 字   预计阅读 1 分钟    次阅读量

Logstash通过网络将syslog消息作为事件读取。使用用 UDPSocket, TCPServer 和

LogStash::Filters::Grok 来实现

配置示例

1
2
3
4
5

input {                                            
    syslog{
        port => "514"
    }
}

配置客户端，在修改linux主机/etc/rsyslog.conf

1

*.* @@host:port

配置说明

参数 说明
* 类型
* 级别
@ udp
@@ tcp
host 可为主机名或ip地址

注：收集到的数据，本身就以及是rsyslog格式了，无需再进行grok

收集到的数据需要对时间进行格式化

官方date插件说明：Date filter plugin

1
2
3
4
5
6
7

filter {
		date {
				match => ["timestamp" , "MMM  dd HH:mm:ss"]
				target => "timestamp"
				"timezone" => "Asia/Shanghai"
		}
}
