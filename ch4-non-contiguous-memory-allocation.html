еҲҶйЎө (Paging) дёҺ еҲҶж®ө (Segmentation) йғҪжҳҜйқһиҝһз»ӯжҖ§еҶ…еӯҳз®ЎзҗҶзҡ„жңәеҲ¶пјҢеҲҶж®өе…Ғи®ёиҝӣзЁӢзҡ„зү©зҗҶең°еқҖз©әй—ҙдёҚиҝһз»ӯпјӣиҖҢеҲҶйЎөеҲҷжҳҜжҜ”еҲҶж®өиҫғжңүдјҳеҠҝзҡ„еҸҰдёҖз§ҚеҶ…еӯҳз®ЎзҗҶж–№жЎҲгҖӮ

еҲҶйЎөзҡ„зү№зӮ№:

  • е°ҶжҜҸдёӘеқ—еҲ’еҲҶдёәеӣәе®ҡзҡ„йЎөгҖӮ
    • еҲ’еҲҶзү©зҗҶеҶ…еӯҳдёәеӣәе®ҡеӨ§е°Ҹ( fixed-sized ) зҡ„еқ— (block) з§°дҪңеё§ ( frame пјү
      • еӨ§е°ҸжҳҜ2зҡ„е№Ӯ, e.g. 512 / 4096 / 8192
    • йҖ»иҫ‘ең°еқҖзӣёеҗҢеӨ§е°Ҹзҡ„зҡ„еқ— (blocks) з§°дҪң (Pages)
      • еӨ§е°ҸжҳҜ2зҡ„е№Ӯ, e.g. 512 / 4096 / 8192
  • жҜҸдёӘж®өйңҖиҰҒдёҖдёӘйЎөиЎЁгҖӮ
  • CPUзҡ„еҶ…еӯҳз®ЎзҗҶеҚ•е…ғйңҖиҰҒеҗҢж—¶ж”ҜжҢҒ еҲҶйЎөе’ҢеҲҶж®өгҖӮ

её§ Frame

зү©зҗҶең°еқҖ пјҲPhysical Addressпјүз©әй—ҙеңЁеҲ’еҲҶдёәиӢҘе№Іеӣәе®ҡеӨ§е°Ҹзҡ„еқ—пјҢз§°дёәеё§гҖӮеё§з”ұдёӨйғЁеҲҶз»„жҲҗпјҢFrame number(f) е’Ң Frame offset(==d==)

  • Frame number(f): иЎЁзӨәзү©зҗҶең°еқҖз©әй—ҙзҡ„её§жүҖйңҖзҡ„дҪҚж•°гҖӮ
  • Frame offset(d): иЎЁзӨәйЎөдёӯзү©зҗҶең°еқҖз©әй—ҙзҡ„йЎөеӨ§е°ҸжҲ–йЎөжҲ–йЎөеҒҸз§»йҮҸзҡ„еӯ—ж•°жүҖйңҖзҡ„дҪҚж•°гҖӮ

еҰӮеӣҫжүҖзӨәпјҢзү©зҗҶең°еқҖжҳҜдёҖдёӘдәҢе…ғз»„ $(f,d)$пјҢйЎөеё§еҚ  $F$ дҪҚпјҢе…ұжңү $2^F$ её§пјӣеҒҸз§»йҮҸ o еҚ дәҶ $S$ дҪҚпјҲдёҖдёӘеё§зҡ„еӨ§е°ҸпјүпјҢе…ұ$2^S$ еӯ—иҠӮ пјҢеҲҷзү©зҗҶең°еқҖ = $2^S\times f+d$ гҖӮ

ең°еқҖи®Ўз®—пјҡ16bitзҡ„зү©зҗҶең°еқҖз©әй—ҙпјҢйЎөеё§еӨ§е°Ҹдёә9bitпјҲ512byteпјү

  • з»ҷеҮәзү©зҗҶең°еқҖ$(f,d) = (3,6)$пјҢжұӮзү©зҗҶең°еқҖзҡ„дҪҚзҪ® гҖӮ
    • $S=9$пјҢ$F=7$пјҢ $f=3$ пјҢ$d=6$
    • еҘ—з”Ёе…¬ејҸеҫ—еҮәпјҢ$2^9\times3+6 = 1542$

  • Page number(p) йҖ»иҫ‘ең°еқҖз©әй—ҙдёӯйЎөжүҖйңҖзҡ„дҪҚж•°

йЎөзҡ„еҜ»еқҖжңәеҲ¶

еңЁеӣҫдёӯеҸҜд»ҘзңӢеҮәпјҢйҖ»иҫ‘ең°еқҖжҳҜдёҖдёӘиҝһз»ӯзҡ„ең°еқҖз©әй—ҙпјҢе№¶дё”з”ұдёҖдёӘдёӘPageз»„жҲҗпјҢйҰ–е…ҲCPUеҜ»еқҖпјҢең°еқҖеҲҶдҪҚдёӨеқ—пјҲдёҖдёӘдәҢе…ғз»„пјү$(p,d)$пјҢpдҪңдёәдёҖдёӘindex еҺ»жҹҘдёҖдёӘpage table (д»ҘйЎөеҸ·дёәзҙўеј•зҡ„еҖјдёәеё§еҸ·) пјҢд»ҘindexдёҺbaseпјҲеҹәең°еқҖпјүдҪңдёәжҹҘжүҫйЎ№жҹҘжүҫеҜ№еә”зҡ„fпјҢ$f + (f)d$ е°ұжүҫеҲ°дәҶеҜ№еә”зҡ„зү©зҗҶең°еқҖгҖӮ

  • йЎөжҳ е°„еҲ°её§
  • йЎөжҳҜиҝһз»ӯзҡ„иҷҡжӢҹеҶ…еӯҳ
  • её§жҳҜйқһиҝһз»ӯзҡ„зү©зҗҶеҶ…еӯҳ
  • дёҚжҳҜжүҖжңүзҡ„йЎөйғҪжңүеҜ№еә”зҡ„её§

    • и„ҸдҪҚ/дҝ®ж”№дҪҚпјҲDirty Bit (D)пјүпјҡжҜҸдёӘPTEйғҪеҢ…еҗ«зҡ„дёҖдёӘдҝ®ж”№дҪҚпјҢиЎЁзӨәйЎөйқўиҮӘеҠ иҪҪеҗҺжҳҜеҗҰе·ІеҶҷе…ҘгҖӮжҳҜеңЁж“ҚдҪңзі»з»ҹдёӯе®ҢжҲҗпјҢиҖҢдёҚжҳҜеңЁзЎ¬д»¶дёӯе®ҢжҲҗзҡ„ж“ҚдҪңгҖӮ
      • 1пјҡиЎЁзӨәж•°жҚ®жҳҜ"и„Ҹзҡ„пјҲdirtyпјү"пјҢеҚіе·ІеҶҷе…ҘгҖӮ
      • 0пјҡж•°жҚ®жҳҜвҖңе№ІеҮҖзҡ„(clean)вҖқпјҢдёҺеҠ иҪҪж—¶зӣёеҗҢгҖӮ
    • еӯҳеңЁдҪҚ/й©»з•ҷдҪҚ (resident bit (R))пјҡжҜҸдёӘPTEйғҪеҢ…еҗ«зҡ„дёҖдёӘй©»з•ҷдҪҚпјҢиЎЁзӨәйҖ»иҫ‘ең°еқҖжҳҜеҗҰжңүдёҖдёӘзү©зҗҶең°еқҖдёҺе…¶зӣёеҜ№еә”гҖӮ
      • 1пјҡеҪ“йҖ»иҫ‘йЎөеңЁзү©зҗҶеҶ…еӯҳдёӯж—¶пјҢиҜҘдҪҚиў«и®ҫзҪ®дёә1гҖӮ
      • 0пјҡеҰӮжһңдёә0пјҢеҲҷи®ҝй—®иҜҘиҷҡжӢҹйЎөйқўе°ҶеҜјиҮҙйЎөйқўй”ҷиҜҜгҖӮ
    • еј•з”ЁдҪҚ (Eviction Bit пјҢIn x86: Reference Bit)пјҢиҝҮеҺ»дёҖж®өж—¶й—ҙеҶ…жҳҜеҗҰжңүеҜ№е…¶еј•з”ЁпјҲжҳҜеҗҰи®ҝй—®иҝҮйЎөеҶ…зҡ„жҹҗдёӘеӯҳеӮЁеҚ•е…ғпјүгҖӮеӯҳеңЁйЎөиЎЁзҡ„з¬¬дәҢдҪҚпјҢеңЁжҜҸж¬Ўи®ҝй—®йЎөйқўж—¶пјҲиҜ»/еҶҷпјүпјҢReference Bitи®ҫзҪ®дёә1гҖӮ

йЎөиЎЁзҡ„иҪ¬жҚў пјҲtranslation processпјү

CPUзҡ„еҶ…еӯҳеҚ•е…ғпјҲMMUпјҢ memory management unitпјүж №жҚ®зЁӢеәҸзҡ„pageзҡ„йЎөеҸ·зҡ„иӢҘе№ІдҪҚ, и®Ўз®—еҮәзҙўеј•еҖјindex, еңЁйЎөиЎЁдёӯжҗңзҙўиҝҷдёӘindex, еҫ—еҲ°зҡ„жҳҜеё§еҸ·, её§еҸ·е’ҢеҺҹжң¬зҡ„offsetз»„жҲҗзү©зҗҶең°еқҖ.

Quick Activity

е…·жңү16дҪҚең°еқҖзҡ„и®Ўз®—жңәзі»з»ҹпјҢзү©зҗҶең°еқҖеӨ§е°Ҹ32KBпјҢжҜҸйЎөеӨ§е°Ҹ1024bitesпјҢзҡ„йҖ»иҫ‘ең°еқҖеҰӮдҪ•иҝӣиЎҢиҪ¬жҚўгҖӮ

16дҪҚең°еқҖжҳҜ0~15пјҢ0~9жҳҜйЎөеҶ…еҒҸз§» d, 10~15жҳҜйЎөеҸ·гҖӮ

йЎөејҸеӯҳеӮЁз®ЎзҗҶжңәеҲ¶еӯҳеңЁзҡ„й—®йўҳ

  • еҶ…еӯҳи®ҝй—®жҖ§иғҪй—®йўҳ(Performance Issues)пјҡеҶ…еӯҳи®ҝй—®жҖ§иғҪпјҢиҷҡжӢҹең°еқҖи®ҝй—®йңҖиҰҒ2ж¬ЎеҶ…еӯҳи®ҝй—®
    • з¬¬дёҖж¬ЎиҺ·еҸ–йЎөиЎЁйЎ№
    • з¬¬дәҢж¬Ўи®ҝй—®ж•°жҚ®
  • йЎөиЎЁеӨ§е°Ҹй—®йўҳпјҡйЎөиЎЁеҸҜд»ҘйқһеёёеӨ§гҖӮ
    • йЎөиЎЁеҸҜиғҪеҫҲеӨ§
      • еҰӮеӣҫе®һдҫӢпјҢ32KеҶ…еӯҳпјҢжҜҸйЎө1KпјҢеҚі32дёӘйЎөиЎЁйЎ№пјҢеҰӮжһңжҜҸйЎ№еҚ 4byteпјҢеҲҷдёә128byteгҖӮ
    • еҜ№дәҺе…·жңү64дҪҚең°еқҖе’Ң1024еӯ—иҠӮйЎөзҡ„жңәеҷЁпјҢйЎөиЎЁзҡ„еӨ§е°ҸжҳҜеӨҡе°‘пјҹ
      • йЎөпјҡ$2^{64}$пјҢйЎөзҡ„еӨ§е°Ҹпјҡ$2^{10}$пјҢйңҖиҰҒе»әз«ӢйЎөиЎЁзҡ„еӨ§е°Ҹпјҡ$2^{64} \div 2^{10} = 2^{54}$

еҰӮдҪ•и§ЈеҶідёҠиҝ°й—®йўҳ(what to do)пјҡ

  • Caching зј“еӯҳпјҢsolutionпјҡеҝ«иЎЁпјҢеҲ©з”Ёзј“еӯҳжңәеҲ¶еҮҸе°‘еҜ№йЎөиЎЁзҡ„и®ҝй—®гҖӮ
  • Indirection й—ҙжҺҘи®ҝй—®пјҢsolutionпјҡеӨҡзә§йЎөиЎЁпјҢйҖҡиҝҮй—ҙжҺҘеј•з”Ёзҡ„ж–№ејҸжқҘеҮҸе°‘йЎөиЎЁзҡ„й•ҝеәҰгҖӮ

еҝ«иЎЁ translation look-aside buffers (TLBs)

еҜ№дәҺдёҠиҝ°зҡ„жҖ§иғҪй—®йўҳпјҢжңүж•Ҳзҡ„и§ЈеҶіж–№жі•жҳҜеҜ№йЎөиЎЁйЎ№пјҲPTEпјүзҡ„й«ҳйҖҹзј“еӯҳпјҢжҲҗдёәеҝ«иЎЁпјҲtranslation look-aside buffer (TLB)пјүгҖӮеҝ«иЎЁе°ұжҳҜе°Ҷиҝ‘жңҹи®ҝй—®иҝҮзҡ„йЎөиЎЁйЎ№еңЁCPUеҶ…йғЁдҪҝз”ЁзЎ¬д»¶зј“еӯҳпјҢеҚіе°Ҷиҝ‘жңҹи®ҝй—®иҝҮзҡ„йЎ№зј“еӯҳеҲ°CPUдёӯпјӣTLBз”ұеҶ…еӯҳз®ЎзҗҶеҚ•е…ғпјҲMMUпјүз®ЎзҗҶ

[жңӘдҪҝз”ЁйЎөиЎЁзҡ„жғ…еҶө]

[TLB mechanics]

еҰӮеӣҫжүҖзӨәпјҢеҰӮжһңTLBе‘ҪдёӯпјҢеҲҷзӣҙжҺҘеҸҜд»ҘиҺ·еҸ–еҲ°зү©зҗҶең°еқҖпјҢеҰӮжһңTLSжңӘе‘ҪдёӯпјҢеҲҷеҗҢж—¶е°ҶеҶ…е®№зј“еӯҳеҲ°CPUдёӯзҡ„TLBгҖӮ

еӨҡзә§йЎөиЎЁ

еӨҡзә§йЎөиЎЁжҳҜйҖҡиҝҮй—ҙжҺҘеј•з”Ёзҡ„ж–№ејҸе°ҶйЎөеҸ·еҲҶжҲҗеӨҡзә§гҖӮеӨҡзә§йЎөиЎЁжҳҜж ‘зҠ¶з»“жһ„пјҢз”ЁдәҺдҝқеӯҳйЎөиЎЁгҖӮ

дҫӢеӯҗпјҢиҖғиҷ‘дёӨдёӘзә§еҲ«зҡ„йЎөиЎЁеҶҚж¬ЎеңЁе…·жңү$2^{12}=4$ KB Page зҡ„32-bitжһ¶дёҠпјҢ

Q1пјҡз¬¬дәҢзә§йЎөйқўиЎЁж јжңүеӨҡе°‘дҪҚпјҹ

  • $4KB\div4B = 1024=2^{10}$

Q2пјҡйЎ¶зә§йЎөиЎЁжңүеӨҡе°‘дҪҚпјҹ

  • $32-12-10=10$

10bits иЎЁзӨә0зә§зҙўеј•пјҢ10bitsиЎЁзӨә1зә§зҙўеј•пјҢ12bitsиЎЁзӨәйЎөйқўеҶ…зҡ„еҒҸз§»йҮҸгҖӮжҜҸдёҖдёӘеӯҗйЎөиЎЁзҡ„ејҖеӨҙдҪңдёәдёҠдёҖдёӘйЎөиЎЁзҡ„йЎөеҸ·зү©зҗҶйЎөеҸ·еЎ«еҶҷеҲ°дёҠдёҖзә§йЎөиЎЁеҪ“дёӯгҖӮ

еӨҡзә§йЎөиЎЁи®Ўз®—йўҳ

дёҖдёӘ32дҪҚж“ҚдҪңзі»з»ҹдёӯиҝӣзЁӢзҡ„иҷҡжӢҹең°еқҖз©әй—ҙеҸҜиҫҫеҲ°4GBпјҢеҒҮи®ҫз”ЁжҲ·ең°еқҖз©әй—ҙдёә2GBпјҢйЎөйқўеӨ§е°Ҹдёә4KBпјҢ

Q1пјҡеҲҷдёҖдёӘиҝӣзЁӢжңҖеӨҡеҸҜд»ҘжңүпјҲпјүйЎөгҖӮ

  • $2G = 2\times 2^{10} \times 2^{10} \times 2^{10} = 2^{31} $вҖӢ
  • $4KB = 2^2 \times 2^{10} = 2^{12}$вҖӢвҖӢ
  • еҚі $2^{31}\div2^{12} = 2^{19}$вҖӢвҖӢ

Q2пјҡиӢҘз”Ё4дёӘеӯ—иҠӮиЎЁзӨәдёҖйЎөзҡ„зү©зҗҶйЎөеҸ·пјҢеҲҷйЎөиЎЁжң¬иә«е°ұеҚ з”ЁпјҲпјүпјҹ

  • $4B\times 2^{19} = 2^2 \times 2^{19} = 2^{21} = 2M $

Q3пјҡеҚійңҖиҰҒпјҲпјүдёӘйЎөйқўеӯҳж”ҫгҖӮ

  • $2M\div4KB = 2^{21}\div2^{12} = 2^{9} = 512$

еҸҚзҪ®йЎөиЎЁ

дҪҝз”ЁйЎөеҜ„еӯҳеҷЁ(Page Registers)пјҢеҸҲеҗҚеҸҚзҪ®йЎөиЎЁ(Inverted page table (IPT))пјҢжҳҜдёәдәҶеҮҸе°‘жүҖеҚ з”ЁеӯҳеӮЁз©әй—ҙзҡ„дёҖз§ҚеҒҡжі•гҖӮдҪҝз”ЁеҸҚзҪ®йЎөиЎЁзҡ„еҺҹеӣ жҳҜеңЁеӨ§ең°еқҖз©әй—ҙ(64-bits)зҡ„зі»з»ҹдёҠпјҢеӨҡзә§йЎөиЎЁдјҡеҸҳеҫ—з№ҒзҗҗпјҢдҫӢеҰӮпјҢйҖ»иҫ‘ең°еқҖз©әй—ҙеўһй•ҝйҖҹеәҰеҝ«дәҺзү©зҗҶз©әй—ҙең°еқҖгҖӮпјҲд»ҘйЎөеё§еҸ·дёәзҙўеј•пјҢйЎөиЎЁеҸ·дёәеҶ…е®№ пјү

жӯӨж—¶дә§з”ҹдёҖдёӘй—®йўҳпјҡеҰӮдҪ•жҹҘжүҫпјҹпјҲframe numberжҳҜзҙўеј•пјҢpagenumberжҳҜеҶ…е®№пјүгҖӮеҹәдәҺе…іиҒ”еӯҳеӮЁеҷЁпјҲassociative memoryпјүзҡ„ж–№жЎҲпјҡ

е…іиҒ”еӯҳеӮЁеҷЁд№ҹеҸҜд»Ҙе№¶иЎҢзҡ„жҹҘжүҫпјҢдёҺеҜ„еӯҳеҷЁеҸҜд»ҘеҗҢж—¶жҜ”иҫғпјҢеҰӮжһңжүҫеҲ°еҢ№й…ҚеҖјпјҢеҲҷиҫ“еҮәдёҺй”®еҜ№еә”зҡ„еҖјгҖӮ

  • еҰӮжһңеё§ж•°иҫғе°‘пјҢйЎөеҜ„еӯҳеҷЁеҸҜд»Ҙиў«ж”ҫзҪ®еңЁе…іиҒ”еҶ…еӯҳдёӯ
  • еңЁе…іиҒ”еҶ…еӯҳдёӯжҹҘжүҫйҖ»иҫ‘йЎөеҸ·

е…іиҒ”еӯҳеӮЁеҷЁзҡ„д»Јд»·пјҡ

еҰӮжһңеё§ж•°иҫғе°‘пјҢйЎөеҜ„еӯҳеҷЁеҸҜд»Ҙиў«ж”ҫзҪ®еңЁе…іиҒ”еҶ…еӯҳдёӯпјӣеңЁе…іиҒ”еҶ…еӯҳдёӯжҹҘжүҫйҖ»иҫ‘йЎөеҸ·пјҡ

  • жҲҗеҠҹпјҢеҲҷжӢҝеҲ°её§еҸ·
  • еӨұиҙҘпјҢд№ҹй”ҷиҜҜејӮеёёпјҲpage faultпјү

йҷҗеҲ¶еӣ зҙ пјҡеҫҲйҡҫеңЁеҚ•дёӘж—¶й’ҹе‘Ёжңҹе®ҢжҲҗпјӣиҖ—з”өгҖӮ

IPTзҡ„е…·дҪ“жҖқи·ҜдёәпјҢдёҚи®©йЎөиЎЁдёҺйҖ»иҫ‘ең°еқҖз©әй—ҙзҡ„еӨ§е°ҸеҜ№еә”пјҢиҖҢжҳҜи®©йЎөиЎЁдёҺзү©зҗҶең°еқҖз©әй—ҙеӨ§е°ҸзӣёеҜ№еә”пјҢжҜҸдёӘеё§йғҪдёҺдёҖдёӘеҜ„еӯҳеҷЁзӣёе…іиҒ”пјҢйҖҡеёёиҜҘеҜ„еӯҳеҷЁеҢ…еҗ«пјҡ

  • еј•з”ЁдҪҚ(Residence bitпјүпјҡж Үи®°иҜҘеё§(Frame) жҳҜеҗҰиў«иҝӣзЁӢеҚ з”ЁгҖӮ
  • еҚ з”ЁйЎөеҸ·(Occupier)пјҡйЎөеё§еҜ№еә”зҡ„йЎөеҸ·
  • дҝқжҠӨдҪҚ(Protection bits)пјҡзәҰе®ҡйЎөзҡ„и®ҝй—®ж–№ејҸ(R/W)

Quick Activity

зі»з»ҹжңү

  • зү©зҗҶеҶ…еӯҳеӨ§е°Ҹпјҡ16MB( $4096 \times 4096 = 4K \times 4K = 16MB $ )
  • йЎөеӨ§е°Ҹдёәпјҡ4KB( $4096bytes = 4KB $вҖӢвҖӢвҖӢвҖӢ)
  • йЎөеё§ж•°дёә4K(4096)гҖӮ
  • еҒҮе®ҡжҜҸдёҖдёӘйЎөеҜ„еӯҳеҷЁ(page resigter)еҚ 8bytesпјҢ $4096 \times 8 = 32Kbytes$гҖӮ

йӮЈд№ҲйЎөеҜ„еӯҳеҷЁеҚ з”Ёзҡ„ејҖй”Җдёәпјҡ $32Kb\div 16Mb \approx 0.2%$вҖӢвҖӢгҖӮжӯӨж—¶е’ҢиҷҡжӢҹең°еқҖжІЎжңүе…іиҒ”дәҶгҖӮеҲӣе»әвҖӢзҡ„иҝӣзЁӢзҡ„зү©зҗҶең°еқҖе°ұдёҺйЎөеҜ„еӯҳеҷЁеҜ№еә”гҖӮ

йЎөеҜ„еӯҳеҷЁж–№жЎҲзҡ„дјҳзјәзӮ№пјҡ

  • дјҳзӮ№пјҡ
    • йЎөиЎЁеӨ§е°ҸзӣёеҜ№дәҺзү©зҗҶеҶ…еӯҳиҖҢиЁҖеҫҲе°Ҹ
    • йЎөиЎЁеӨ§е°ҸдёҺйҖ»иҫ‘пјҲиҷҡжӢҹпјүең°еқҖж— е…і
  • зјәзӮ№пјҡ
    • йЎөиЎЁдҝЎжҒҜеҜ№и°ғеҗҺпјҢйңҖиҰҒеё§еҸ·еҸҜд»ҘжүҫеҲ°йЎөеҸ·
    • еңЁйЎөеҜ„еӯҳеҷЁдёӯжҗңзҙўйҖ»иҫ‘ең°еқҖдёӯзҡ„йЎөеҸ·

еҹәдәҺhashи®Ўз®—зҡ„еҸҚеҗ‘йЎөиЎЁ hash table

hash tableпјҢдёҖдёӘж•°еӯҰи®Ўз®—зҡ„ж–№жі•гҖӮйҖҡиҝҮhashи®Ўз®—иҫ“е…Ҙдёәpage numberпјҢиҫ“еҮәдёә frame numberгҖӮз”ұеӣҫжүҖзӨә

  • Virtual Page Number (VPN): p, q
  • Page Frame Number (PFN): r
  • Offset: d
  • Hash Function: h(x)
  • Hashed Page Table with schema (key, VPN, PFN, Pointer to next entry with key) жҜҸдёҖдёӘйЎөиЎЁйЎ№

hash tableж–№ејҸзҡ„е·ҘдҪңжөҒ

  • ж“ҚдҪңзі»з»ҹд»ҺCPUжӢҝеҲ° иҷҡжӢҹеҶ…еӯҳ pпјҢе№¶жү§иЎҢ hпјҲpпјү д»ҘиҺ·еҸ– same_keyгҖӮ

  • ж“ҚдҪңзі»з»ҹжҹҘжүҫе“ҲеёҢйЎөиЎЁдёӯзҡ„з¬¬дёҖдёӘжқЎзӣ®пјҢе…¶дёӯ key = same_keyпјҢе№¶ж №жҚ®з¬¬дёҖдёӘйЎөиЎЁзҡ„ иҷҡжӢҹеҶ…еӯҳйЎөеҸ· pгҖӮ(ж №жҚ®з¬¬дёҖдёӘйЎ№дёӯзҡ„жҢҮй’ҲжқҘжҹҘжүҫз¬¬дәҢдёӘйЎ№гҖӮе®ғзҹҘйҒ“з¬¬дәҢдёӘйЎ№е…·жңүзӣёеҗҢзҡ„й”® = same_keyпјҢеӣ дёәеӨҡзә§йЎөиЎЁе·ҘдҪңжөҒжҳҜеҰӮжӯӨгҖӮж“ҚдҪңзі»з»ҹж №жҚ®з¬¬дәҢйЎ№зҡ„йЎөиЎЁеҸ·еӯ—ж®өжЈҖжҹҘpгҖӮ

  • ж“ҚдҪңзі»з»ҹд»Һз¬¬дәҢдёӘжқЎзӣ®дёӯиҺ·еҸ–её§еҸ· rгҖӮr еҜ№еә”дәҺиҷҡжӢҹйЎөеҸ· p зҡ„жӯЈзЎ®зү©зҗҶеё§еҸ·пјӣж“ҚдҪңзі»з»ҹдҪҝз”Ё йЎөеё§r + еҒҸз§»йҮҸ d еңЁзү©зҗҶеҶ…еӯҳдёӯжҹҘжүҫе®ғжғіиҰҒзҡ„зү©зҗҶең°еқҖгҖӮ

з”ұеӣҫеҸҜд»ҘзңӢеҮәпјҢдёәдәҶжҸҗй«ҳж•ҲзҺҮпјҢеҸҜд»ҘеҜ№hashеҮҪж•°еўһеҠ дёҖдёӘеҸӮж•° pid пјҢеҸҜд»ҘдҪңдёәиҫ“е…ҘпјҢжқҘи®ҫи®ЎдёҖдёӘhashеҮҪж•°пјҢз®—еҮәеҜ№еә”зҡ„frame numberгҖӮиҝҷз§ҚеҸҜд»ҘеҫҲеҘҪзҡ„и§ЈеҶіжҳ е°„зҡ„ејҖй”ҖгҖӮ

ж®өйЎөејҸеӯҳеӮЁз®ЎзҗҶ

ж®өйЎөејҸеӯҳеӮЁз®ЎзҗҶжңәеҲ¶ ( Segmented Paging )пјҢжҳҜд»ҺйЎөејҸдёҺж®өејҸдёӨз§ҚжҠҖжңҜдёӯиҺ·еҫ—жңҖдҪізү№жҖ§пјҢзәҜеҲҶж®өдёҚжҳҜеҫҲжөҒиЎҢпјҢе№¶дё”еңЁи®ёеӨҡж“ҚдҪңзі»з»ҹдёӯйғҪжІЎжңүдҪҝз”ЁгҖӮдҪҶжҳҜпјҢеҲҶж®өеҸҜд»ҘдёҺеҲҶйЎөзӣёз»“еҗҲгҖӮж®өејҸеӯҳеӮЁеңЁеҶ…еӯҳдҝқжҠӨж–№йқўжңүдјҳеҠҝпјҢйЎөејҸеӯҳеӮЁеңЁеҶ…еӯҳеҲ©з”Ёе’ҢиҪ¬з§»еҲ°еҗҺеӨҮеӯҳеӮЁж–№йқўжңүдјҳеҠҝгҖӮ

ж®өйЎөејҸеӯҳеӮЁз®ЎзҗҶжңәеҲ¶жҳҜеңЁж®өејҸеӯҳеӮЁзҡ„еҹәзЎҖдёҠз»ҷжҜҸдёҖдёӘж®өеҠ дёҖзә§йЎөиЎЁгҖӮеҚіпјҢдё»еӯҳиў«еҲ’еҲҶдёәеҸҜеҸҳеӨ§е°Ҹзҡ„ж®өпјҢиҝҷдәӣж®өеҸҲиў«иҝӣдёҖжӯҘеҲ’еҲҶдёәеӣәе®ҡеӨ§е°Ҹзҡ„йЎөпјҢжӯӨж—¶е°ұжңүдёүдёӘе®ҡд№үиҜҚDefinitions

  • ж®өеҸ· **Segment Number **
  • йЎөеҸ· **Page Number **
  • йЎөеҒҸз§» **Page Offset **

еҰӮеӣҫжүҖзӨәпјҢйҖҡиҝҮдёәжҜҸдёӘж®өеҲӣе»әйЎөиЎЁпјҢеҸҜд»ҘеҮҸе°ҸйЎөиЎЁзҡ„еӨ§е°ҸпјҲиҰҒе®һзҺ°иҝҷдёҖзӮ№пјҢйңҖиҰҒзЎ¬д»¶ж”ҜжҢҒпјүпјҢCPUжҸҗдҫӣзҡ„ең°еқҖзҺ°еңЁе°Ҷиў«еҲ’еҲҶдёәж®өеҸ·гҖҒйЎөеҸ·е’ҢеҒҸз§»йҮҸгҖӮ

ж®өйЎөејҸеӯҳеӮЁжңәеҲ¶е·ҘдҪңжөҒеҰӮеӣҫжүҖзӨәпјҡ

CPUз”ҹжҲҗдёҖдёӘйҖ»иҫ‘ең°еқҖеҲҶдёәдёӨйғЁеҲҶпјҡж®өеҸ·е’Ңж®өеҒҸз§»йҮҸгҖӮж®өеҒҸз§»еҝ…йЎ»е°ҸдәҺж®өйҷҗеҲ¶гҖӮеҒҸз§»йҮҸеҸҲеҲҶдёәйЎөз Ғе’ҢйЎөеҒҸз§»гҖӮиҰҒжҳ е°„йЎөиЎЁдёӯзҡ„зЎ®еҲҮйЎөз ҒпјҢиҜ·е°ҶйЎөз Ғж·»еҠ еҲ°йЎөиЎЁеә“дёӯгҖӮеёҰжңүйЎөеё§еҸ·+йЎөеҒҸз§»з»„жҲҗдәҶе®һйҷ…иў«жҳ е°„зҡ„зү©зҗҶең°еқҖгҖӮ
