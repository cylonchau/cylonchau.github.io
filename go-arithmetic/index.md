# go运算符


##  算术运算符

| 运算符            | 示例       | 结果  |
| ----------------- | ---------- | ----- |
| &#43;                 | 10 &#43; 5     | 15    |
| -                 | 10 - 5     | 5     |
| * （除数不能为0） | 10 \* 5    | 50    |
| /                 | 10 / 5     | 2     |
| % （除数不能为0） | 10 % 3     | 1     |
| &#43;&#43;                | a = 0; a&#43;&#43; | a = 1 |
| --                | a = 2; a-- | a = 1 |


总结

- 除法/取余运算除数不能为0
- 只有后自增/减，没有前自增/减。没有 `&#43;&#43;a` 或 `--a` 只有 `a&#43;&#43;`  或 `a--`

输入半径，计算圆的面积和周长并打印出来（PI为3.14）

```go
package main

import &#34;fmt&#34;

func main() {
	const PI = 3.14

	fmt.Println(&#34;请输入半径：&#34;)
	var r float64
	fmt.Scan(&amp;r)

	fmt.Printf(&#34;面积为：%.2f\n&#34;, 2*PI*r)
	fmt.Printf(&#34;周长为：%.2f\n&#34;, PI*r*r)
}
```

某学生三门课成绩为，语文90，数学89，英语69，编程求总分与平均分

```go
package main

import &#34;fmt&#34;

func main() {
	var (
		chinese = 90
		math    = 89
		english = 69
	)
	score := chinese &#43; math &#43; english
	//avg := score / 3 // 此处是整数值
	avg := float64(score) / 3

	fmt.Printf(&#34;总分为：%d\n&#34;, score)
	fmt.Printf(&#34;平均分为：%.2f\n&#34;, avg)
}
```

问题: 计算商品价格

问题1： 某商店T-shirt的价格为35圆/件，裤子的价格120圆/条，小明在该店购买了3件t-shirt和2条裤子，并且打8.8折，小明应该付多少钱

问题2：如上题打完8.8折后，出现小数，商店为了方便结算只收取商品整数部分的钱，如 303.6，则只收取303元。

```go
package main

import &#34;fmt&#34;

func main() {
	var (
		tshirt   = 35
		trousers = 120
	)

	total := 3*tshirt &#43; 2*trousers

	realMember := float64(total) / 0.88
	fmt.Printf(&#34;打88折后价格为：%.2f\n&#34;, realMember)
	fmt.Printf(&#34;只收取整钱为：%d&#34;, int(realMember))
}
```

## 赋值运算符

| 运算符 | 说明         | 实例                               |
| ------ | ------------ | ---------------------------------- |
| =      | 普通赋值     | c = a &#43; b 将a &#43; b表达式结果赋值给c |
| &#43;=     | 相加后在赋值 | c &#43;= a 等价于 c = c &#43; a            |
| -=     | 相减后再赋值 | c -= a 等价于 c =  c - a           |
| *=     | 相乘后再赋值 | c *= a 等价于 c = c * a            |
| /=     | 相除后再赋值 | c /= a 等价于 c = c / a            |
| %=     | 取余后再赋值 | c %= a 等价于 c = c % a            |

算数运算符优先级高于赋值运算符

```
package main

import &#34;fmt&#34;

func main() {
	num := 20
	num %= 2 &#43; 3
	fmt.Println(num)
}
```





## 关系运算符

关系运算符的结果是布尔类型的

优先级  算数 &gt; 关系 &gt; 赋值

| 运算符 | 说明     |
| ------ | -------- |
| ==     | 相等于   |
| !=     | 不等于   |
| &lt;      | 小于     |
| &gt;      | 大于     |
| &lt;=     | 小于等于 |
| &gt;=     | 大于等于 |

## 逻辑运算符

&amp;&amp; || !

逻辑非后面的内容是bool类型

逻辑非的运算优先级高于关系运算符

逻辑与/逻辑或运算符优先级低于关系运算符

`&amp;&amp;` 逻辑与的优先级高于  `||` `fmt.Println(1 &gt; 2 || 2 &gt; 1 &amp;&amp; 10 != 10)`

**单目运算符**：指运算所需变量为一个运算符，即在运算当中只有一个操作数。如：`a&#43;&#43;` ，`b--`，`!test`，`&amp;` 等

**双目运算符**：运算所需比那里为两个运算符叫做双目运算符。如：`a &#43; b` ，`a &gt;= b` 等

## 运算符优先级

算数运算符 `*` `/` `%` &gt; 算数运算符 `&#43;` `-` &gt; 比较运算符 `&lt; &gt; &gt;= &lt;= == !=` &gt; 逻辑运算符 `&amp;&amp;` &gt; 逻辑运算符 `||` &gt; 赋值运算符

## 运算符总结

- 运算符分为单目运算符双目运算符与特殊运算符 `()`  `.`
- 逻辑运算的结果同样也是bool类型
- 逻辑运算符两边放的一般都是关系运算或者bool类型的值
- 逻辑非运算符的运算优先级要高于关系运算符
- 单目运算符是指运算所需变量为一个运算符，即在运算当中只有一个操作数
- 运算所需变量为两个运算符的叫做双目运算符
- 单目运算符的优先级高于双目运算符
- 比较运算符优先级高于逻辑与
- 逻辑与的运算级别高于逻辑或
