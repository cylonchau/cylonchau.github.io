复杂查询

query ： 相当于查询

  • bool： 组合条件查询
    • must： 条件必须满足，相当于and match_all 查询所有
    • should 条件可以满足也可以不满足，相当于or
    • must_not 条件不需要满足，相当于not

range 范围过滤

  • gt > 大于
  • lt < 小于
  • gte >= 大于等于
  • lte <= 小于等于

curl http://172.18.3.106:9200/store/books/_search -H "Content-type: application/json" -d '{
    "query": {
        "range" : {
            "price":{
                "gte" :10,
                "lt" : 90
            }
        }
    }
}'
